BlenderCH5右下Properties Panel

什麼是 Properties Panel？

位置：Blender 介面右下角（預設）。
功能：用來管理場景、物件和材質的各種屬性，提供對細節的全面控制。
特色：包含多個標籤（Tabs），每個標籤都有對應的圖標，涵蓋渲染、場景設定、材質等內容。

Blender 屬性面板（Properties Panel）完整詳細解析

1. Render（渲染面板）

圖標：相機圖案
用途：設置渲染細節，控制畫質、速度和最終效果。

1.1 Render Engine（渲染引擎）
  • 選項：
    • Eevee：實時渲染引擎，快速但適合草稿渲染或簡單動畫。
    • Cycles：物理基礎路徑追蹤引擎，適合高品質靜態圖或影片輸出。
    • Workbench：用於快速預覽，專注於視覺化材質和幾何形狀。
1.2 Sampling（取樣設定）
  • Viewport（視窗取樣）：設定即時視窗的取樣數（影響預覽品質和速度）。
  • Render（最終取樣）：控制最終渲染取樣數。數值越高，畫質越細緻，但時間越長。
    • Denoising（去噪）：使用 AI 或算數方法消除渲染噪點（支援 OptiX 和 OpenImageDenoise）。
1.3 Output Settings（輸出設置）
  • Resolution（解析度）：設定輸出的寬高，例如 1920x1080（Full HD）。
  • Frame Rate（幀率）：設置動畫每秒顯示的幀數，例如 24fps（電影標準）或 30fps（常見動畫標準）。
  • Tile Size（區塊大小）：分塊渲染設定，適合多核 CPU 或 GPU 優化。
1.4 Light Paths（光線路徑）
  • Max Bounces：調整光線反射次數，控制全域光照效果（例如鏡面反射或折射光線）。
  • Transparency Bounces：透明物體的光線追蹤深度。
1.5 Motion Blur（動態模糊）
  • Enable Motion Blur：開啟動態模糊效果，適用於快速移動的物體。
  • Shutter：設定快門速度，影響模糊程度。

2. Output（輸出面板）

圖標：文件夾圖案
用途：設置渲染輸出位置和格式。

2.1 File Path（文件路徑）
  • 輸出目錄：指定渲染結果的保存路徑（例如 /renders/）。
2.2 File Format（文件格式）
  • 靜態圖格式：如 PNG、JPEG、EXR、TIFF 等。
    • Color Depth：設定位深（8-bit、16-bit、32-bit），影響顏色範圍和細節。
  • 動畫格式：如 AVI、MP4、MOV。
    • Encoding：控制壓縮設定（例如 H.264、HEVC）。
2.3 Overwrite / Placeholder
  • Overwrite：啟用後，會覆蓋已有的輸出文件。
  • Placeholder：為尚未完成的幀生成占位文件，適合分布式渲染。

3. View Layer（視圖層面板）

圖標：堆疊圖案
用途：分層渲染管理。

3.1 Layer Includes（視圖層包含）
  • Include：啟用物件或元素是否參與當前層的渲染（例如場景中的燈光、相機）。
  • Mask Layers：指定哪些物件作為遮罩使用。
3.2 Render Passes（渲染通道）
  • Combined（綜合圖層）：包含所有場景元素的通道。
  • Diffuse、Glossy、Transmission：分別輸出漫反射、高光和透射效果。
  • Z Depth：場景深度信息，用於後期合成。
3.3 Layer Overrides（層覆寫）
  • 全域改變屬性，例如替換所有物件材質以加速測試渲染。

4. Scene（場景面板）

圖標：球體與方塊圖案
用途：控制場景的全域屬性。

4.1 Units（單位設定）
  • 單位類型：如公制（Meters）或英制（Feet）。
  • Scale（縮放比例）：影響場景的物理尺寸。
4.2 Keying Sets（關鍵幀設置）
  • 快速選擇哪些屬性會記錄為關鍵幀，例如位置、旋轉或縮放。
4.3 Audio（音訊設置）
  • 同步模式：例如 AV-sync（聲畫同步）。

5. World（世界面板）

圖標：地球圖案
用途：控制場景背景和全域光。

5.1 Surface（表面設置）
  • 背景顏色：簡單的單色背景。
  • 環境貼圖：用 HDRI 圖片作為環境背景。
5.2 Ambient Occlusion（環境遮蔽）
  • 增強陰影效果，模擬更真實的光照環境。
5.3 Mist（霧效果）
  • 控制遠景的霧化程度，常用於營造深度感。

6. Material（材質面板）

圖標：球體圖案
用途：設置和管理物件的材質。

6.1 Base Color（基礎顏色）
  • 設置材質的主要顏色，可連接貼圖。
6.2 Specular（高光）
  • 控制材質表面的反射強度。
6.3 Roughness（粗糙度）
  • 表面光滑程度，值越高越散射光線。
6.4 Use Nodes（使用節點）
  • 啟用節點編輯，支援創建自定義材質。

7. Modifiers（修改器面板）

圖標：扳手圖案
用途：添加非破壞性修改。

7.1 Subdivision Surface
  • 細分模型，增加平滑度。
7.2 Mirror
  • 鏡像複製物件。
7.3 Array

沿特定方向排列多個物件。

8. Texture（貼圖屬性面板）

圖標：棋盤格圖案
用途：用於管理材質中的貼圖，特別是在沒有使用節點時手動分配貼圖資源。

8.1 Texture Type（貼圖類型）

  • Image or Movie：載入靜態圖片（如 PNG、JPEG）或影片作為貼圖。
    • Image Settings：選擇圖片文件，設置影像插值方式（如 Linear 或 Closest）。
    • Auto Refresh：適用於影片貼圖，確保每幀正確更新。
  • Procedural Textures（程序貼圖）：內建多種程序化貼圖生成，如：
    • Noise（雜訊）：用於隨機紋理效果，如雲層或大理石紋路。
    • Voronoi：生成有機的細胞結構紋理，常用於模擬石材或裂縫。

8.2 Mapping（貼圖映射）

  • Coordinates（座標系統）：選擇貼圖的映射方式：
    • UV：使用物件的 UV 展開座標（最常見）。
    • Object：基於物件空間自動生成。
    • Generated：使用 Blender 自動生成的全局座標。
  • Translation / Rotation / Scale（位移/旋轉/縮放）：精細調整貼圖位置和大小。

8.3 Influence（影響權重）

  • Diffuse：控制貼圖對顏色的影響強度。
  • Normal：用於法線貼圖，影響表面凹凸效果。
  • Specular：調整貼圖對高光的影響，增強金屬質感。

9. Particles（粒子屬性面板）

圖標：火花圖案
用途：用於生成粒子系統，如煙霧、火花、毛髮、草地等。

9.1 Basic Settings（基本設定）

  • Type（粒子類型）：分為兩種：
    • Emitter：用於生成粒子，如煙霧、火花、塵埃等動態粒子。
    • Hair：用於模擬毛髮、草地等靜態粒子效果。
  • Number（數量）：設定粒子生成的數量。
  • Lifetime（壽命）：控制粒子存在的幀數。

9.2 Physics（物理屬性）

  • Emitter（發射器設定）：
    • Normal：粒子生成的初速度。
    • Randomize：加入隨機性，模擬更自然的效果。
  • Force Field：與場景中的力場交互，例如：
    • Wind：讓粒子受風力影響移動。
    • Gravity：設定粒子受重力影響的程度。

9.3 Render（粒子渲染）

  • Render As：選擇粒子的外觀：
    • Halo：光點效果，適合星星或霧氣。
    • Object：將場景中的物件複製到每個粒子上。
    • Path：用於 Hair 粒子，生成連續的路徑。

10. Physics（物理面板）

圖標：彈簧圖案
用途：設置物理模擬效果，如布料、液體、剛體、柔體等。

10.1 Cloth（布料）

  • Quality Steps：模擬精度，數值越高效果越真實。
  • Pin Group：選擇頂點組，用於固定布料的特定部分（如旗子的固定點）。
  • Collision（碰撞）：
    • Self Collision：避免布料自身穿透。

10.2 Fluid（流體）

  • Type：設置為液體或煙霧：
    • Domain（域）：定義模擬的邊界空間。
    • Flow（流體來源）：指定流體的生成位置。
  • Viscosity（黏滯性）：模擬液體的濃稠程度。

10.3 Rigid Body（剛體）

  • Type：分為「Active（動態）」和「Passive（靜態）」。
  • Mass（質量）：控制物體的重量，影響模擬效果。
  • Constraints（約束）：限制剛體的運動，例如鉸鏈或彈簧。

11. Constraints（約束面板）

圖標：鎖鏈圖案
用途：用於限制物件的運動方式。

11.1 常用約束類型

  • Copy Location（複製位置）：讓一個物件的位移跟隨另一個物件。
  • Track To（追蹤到）：讓物件始終朝向目標，例如攝影機追蹤物體。
  • Limit Rotation（限制旋轉）：限制物件的旋轉軸向。

12. Shader Editor（著色節點面板）

圖標：節點圖案
用途：在節點模式下設置物件材質。

13. Scripting（腳本面板）

圖標：腳本圖案
用途：使用 Python 進行自動化操作。

13.1 Script 編輯器

 

    • 提供編碼區塊，支援多行程式語法。

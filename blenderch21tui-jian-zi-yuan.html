BlenderCH21角色

一、角色建模基礎

角色建模的目的是構建角色的初始形狀。從簡單的幾何形狀開始，逐步細化到完成的模型。

1. 預備工作

  1. 設定參考圖像
    • 在 Blender 中，切換到「前視圖」（按數字鍵 1），進入正交模式（按數字鍵 5）。
    • 導入參考圖像：點擊 Shift + A > Image > Reference，選擇角色的正面和側面圖片。
    • 將圖像分別對齊到 Z 軸（正面）和 X 軸（側面），確保兩張圖片比例一致。
  2. 設置單位和比例
    • 在右側屬性欄中，點擊「場景屬性（Scene Properties）」> 單位（Units），設置為「公制（Metric）」。
    • 通常，角色的高度設為 1.8 公尺（成人）。

2. 建立基礎形狀

  1. 添加基本幾何

    • 按 Shift + A 添加基本幾何體（例如立方體或圓柱體）。
    • 通過縮放（S 鍵）和移動（G 鍵）調整大致的頭部、軀幹、手臂和腿部的比例。

  2. 使用鏡像修改器（Mirror Modifier）

    • 為角色建模只需創建一側，使用鏡像功能自動生成對稱部分。
    • 添加修改器：右側屬性欄 > 「修改器」> Add Modifier > Mirror。
    • 將鏡像軸設置為 X 軸，確保中心線對齊角色的中線。

  3. 調整比例

    • 參考「8頭身比例法」：
      • 頭部：1個頭部的高度。
      • 頸部到腰部：3個頭部的高度。
      • 腿部：4個頭部的高度。

3. 擴展和細化模型

  1. 細分幾何

    • 點擊 Ctrl + R 使用 Loop Cut，增加網格細分。
    • 根據角色的特徵，增加額外的細分以便進一步雕刻。

  2. 手動調整頂點

    • 切換到編輯模式（Tab 鍵），選擇頂點（Vertex Selection）。
    • 使用移動（G 鍵）和縮放（S 鍵）調整模型的形狀。

二、雕刻工具使用

雕刻是用來添加模型細節的核心功能。Blender 提供多種雕刻筆刷和實用工具。

1. 切換到雕刻模式

  1. 在物件模式中，選中模型。
  2. 按 Tab 或從左上角模式菜單切換到「雕刻模式（Sculpt Mode）」。
  3. 確保模型的多邊形數量足夠（應用細分曲面或啟用動態拓撲，見下文）。

2. 使用雕刻筆刷

Blender 提供多達 20 多種筆刷。以下是常用筆刷的介紹及應用：

  1. Draw（繪畫筆刷）

    • 功能：添加或移除模型表面的小凸起或凹陷。
    • 用途：雕刻皮膚細節、肌肉線條。
    • 調整：改變筆刷大小（F 鍵）和強度（Shift + F 鍵）。

  2. Smooth（平滑筆刷）

    • 功能：平滑模型表面，消除不規則的細節。
    • 用途：修正雕刻時的多餘細節。
    • 快捷鍵：按住 Shift 在任何筆刷模式下臨時切換到平滑。

  3. Crease（摺痕筆刷）

    • 功能：創建銳利的摺痕或邊緣。
    • 用途：雕刻眼睛周圍的皺紋或衣服的褶皺。

  4. Grab（抓取筆刷）

    • 功能：移動大範圍的幾何。
    • 用途：快速調整角色比例，例如延長手臂或腿部。

  5. Inflate（膨脹筆刷）

    • 功能：使模型局部區域向外膨脹。
    • 用途：雕刻肌肉或角色的臉頰。

3. 動態拓撲（Dynamic Topology）

  1. 開啟動態拓撲
    • 在雕刻模式下，右側工具欄中啟用「動態拓撲」。
    • 選擇細分方式：細分-摺疊（Subdivision Collapse）適合創建更精細的細節。
  2. 優化性能
    • 選擇適當的解析度（Resolution），通常設為 10-20。
    • 經常檢查多邊形數量，避免過多頂點導致卡頓。

三、細節處理

細節處理是讓模型更具真實感的關鍵步驟。

1. 細分曲面

  1. 添加細分曲面修改器
    • 選中模型，進入「修改器」面板。
    • 添加 Subdivision Surface 修改器，設置細分級別為 2 或 3。
  2. 應用修改器
    • 當模型基礎完成且雕刻細節已定型時，應用修改器（Apply），以便進一步雕刻。

2. 增加微細細節

  1. 使用細節雕刻筆刷
    • 例如，使用 Crease（摺痕筆刷）雕刻角色臉部的細紋，或用 Draw 筆刷加強肌肉的線條感。
  2. 手繪貼圖
    • 切換到 Texture Paint 模式，在模型表面繪製紋理，添加逼真的皮膚細節或衣物紋理。

3. 模擬布料與配件

  1. 模擬布料
    • 使用 Blender 的布料模擬器創建角色的衣服。
    • 調整重力和布料屬性以模擬真實的褶皺效果。
  2. 添加配件
    • 使用獨立建模或雕刻技術創建角色的配件，如帽子、鞋子或武器。

四、拓撲重建（Retopology）

在雕刻模式下，模型的多邊形數量可能非常高，這對動畫或遊戲來說並不實用。拓撲重建能有效降低多邊形數量並優化網格結構。

1. 手動拓撲重建

  1. 添加新網格

    • 創建新物件，添加一個簡單的平面（Plane）。
    • 進入編輯模式（Tab鍵），將平面縮放至適合角色的比例。

  2. 啟用快照功能（Snap）

    • 在界面頂部啟用「快照到面（Snap to Face）」。
    • 開啟「投射到其他物體（Project Individual Elements on Surface）」。

  3. 手動重新拓撲

    • 選擇角色模型作為參考，在其表面上逐步構建新的低多邊形網格。
    • 使用 Ctrl + R 添加邊環（Loop Cut），並將頂點手動調整到合理位置。
    • 確保拓撲結構遵循動畫要求，例如保持環形結構（Loops）在眼睛、嘴巴和關節處。

2. 自動拓撲重建

如果手動拓撲過於耗時，可以嘗試使用 Blender 自帶的自動拓撲工具：

  1. 啟用動態重建（Remesh）
    • 在雕刻模式下，使用右側工具欄中的「重構（Remesh）」功能。
    • 設置適當的解析度（Voxel Size），例如 0.01 - 0.05，視模型細節而定。
  2. 檢查結果
    • 自動拓撲可能會導致某些部位的細節丟失，需要手動微調。

五、UV 展開

UV 展開是將 3D 模型的表面「展平」成 2D，便於應用材質和貼圖。

1. 自動展開 UV

  1. 進入編輯模式（Tab鍵），選中模型。
  2. 使用 U 快捷鍵，選擇「Smart UV Project」。
    • 此方法適合初學者，可快速生成合理的 UV 展開，但控制力較低。

2. 手動展開 UV

  1. 創建縫線（Seam）
    • 在編輯模式中，切換到邊緣選擇（Edge Select）。
    • 選中需要切割的邊（如模型背部或非顯眼處），按 Ctrl + E > 「標記縫線（Mark Seam）」。
  2. 展開 UV
    • 按 U > 「展開（Unwrap）」，查看展開效果。
  3. 檢查 UV 分佈
    • 切換到 UV 編輯工作區，確保展開的 UV 峰值均勻分布且無明顯拉伸。

六、材質與貼圖

材質和貼圖能賦予角色逼真的外觀，從皮膚紋理到衣物細節。

1. 基本材質設置

  1. 進入材質屬性（右側屬性面板中的球體圖標）。
  2. 點擊「新建（New）」創建材質。
  3. 設置基礎顏色
    • 在基礎著色器（Principled BSDF）中調整 Base Color。
    • 添加粗糙度（Roughness）和金屬度（Metallic）來模擬不同材質效果。

2. 使用貼圖

  1. 添加貼圖
    • 導入外部紋理貼圖（如皮膚、衣物的圖像），並應用到 Base Color 中。
  2. 連接紋理節點
    • 切換到「著色工作區（Shading Workspace）」。
    • 添加「圖像紋理（Image Texture）」節點，連接到基礎著色器的 Base Color。
  3. 檢查貼圖
    • 在 UV 編輯器中查看貼圖分佈，確保無明顯的拉伸或錯位。

3. 法線貼圖與置換貼圖

  1. 法線貼圖（Normal Map）
    • 添加法線貼圖節點，用於模擬表面細節（如皺紋、紋理）。
    • 連接到基礎著色器的「Normal」插槽。
  2. 置換貼圖（Displacement Map）
    • 如果需要更真實的細節，可以使用置換貼圖來改變模型表面幾何。

七、角色綁定與骨骼系統（Rigging）

角色綁定是為模型添加骨骼，用於動畫控制。

1. 添加骨骼（Armature）

  1. 按 Shift + A > Armature > Single Bone，添加一根骨骼。
  2. 進入骨骼編輯模式（Edit Mode），根據角色的身體結構調整骨骼位置。
    • 例如，添加脊椎、手臂、腿部和頭部骨骼。

2. 自動權重綁定

  1. 選中角色模型，然後按住 Shift 選中骨骼。
  2. 按 Ctrl + P > 選擇「自動權重（With Automatic Weights）」。
    • 這會自動將模型的頂點分配給相應的骨骼。

3. 測試綁定效果

  1. 切換到姿勢模式（Pose Mode）。
  2. 選中骨骼並進行旋轉（R鍵）或移動（G鍵），檢查模型是否跟隨骨骼變形。

八、燈光與渲染

完成模型後，可以添加燈光和相機，進行最終的渲染。

1. 添加燈光

  1. 按 Shift + A > Light > Point/Spot/Sun，選擇適合的燈光類型。
  2. 調整燈光強度和位置，確保角色面部和細節能夠清晰呈現。

2. 調整渲染設置

  1. 進入渲染屬性（右側屬性面板中的相機圖標）。
  2. 渲染引擎選擇：
    • Eevee：適合快速渲染和預覽。
    • Cycles：適合高品質渲染，支持全局光照和真實反射。
  3. 調整分辨率與採樣數量：
    • 分辨率：1920x1080 或更高。
    • 採樣：Eevee 建議 64-128，Cycles 建議 200-500。

3. 最終渲染

 

      1. 設置好相機角度（按 N 調整 View > Lock Camera to View）。
      2. 按 F12 渲染圖片。
      3. 渲染完成後，保存結果（Ctrl + S 或 File > Save As）。

九、進階拓撲與網格優化

1. 確保動畫友好的拓撲

拓撲重建時需要考慮模型的動畫效果，以下部位的拓撲需特別注意：

  • 面部：眼睛、嘴巴周圍應該有同心環形結構（Loops），用於表情動畫。
  • 關節：膝蓋、肘部等關節處應有額外的邊環以支持彎曲和變形。

2. 清理多餘的幾何

  1. 切換到編輯模式，選擇所有頂點（A 鍵）。
  2. 在頂點菜單中選擇「合併重疊頂點（Merge by Distance）」，刪除重疊頂點。
  3. 使用 Ctrl + T 將非三角形面轉為三角形（適合遊戲引擎使用）。

3. 減少網格密度

  • 對於高多邊形模型，可以使用「Decimate Modifier」減少面數。
  • 減少後檢查模型細節是否受到影響，必要時手動修復。

十、表情綁定與形變（Shape Keys）

如果角色需要有表情變化，可以使用形變鍵（Shape Keys）製作面部表情。

1. 添加形變鍵

  1. 在物件模式下，選中角色模型。
  2. 在屬性欄中打開「數據屬性（Object Data Properties）」> Shape Keys。
  3. 點擊「+」號，創建基礎形狀（Basis）。
  4. 再次點擊「+」，創建新的形變鍵，命名為例如「Smile」。

2. 編輯形變

  1. 切換到編輯模式，手動調整頂點，創建表情（如嘴角上揚）。
  2. 返回物件模式，調整形變滑桿（Value），預覽表情效果。

3. 配合骨骼使用

  • 形變鍵可以與骨骼結合，例如角色笑時張嘴的幅度與形變鍵同步。

十一、進階材質與貼圖

1. 使用多層貼圖

  1. 皮膚細節層次
    • 基礎貼圖（Base Color）：角色皮膚的主要顏色。
    • 法線貼圖（Normal Map）：添加毛孔、皺紋等細節。
    • 高光貼圖（Specular Map）：模擬皮膚的光澤。
  2. 疊加其他材質
    • 例如添加斑點、疤痕或泥土效果，可用透明度貼圖（Alpha Map）疊加到皮膚上。

2. 使用貼圖繪製工具

  1. 在「紋理繪製模式（Texture Paint）」中，可以直接在模型上畫出細節，例如斑點、化妝、傷疤。
  2. 配合筆刷（Brush）和遮罩（Mask）功能，控制細節繪製範圍。

十二、進階渲染與後期處理

1. 使用 HDR 環境光

  1. 添加 HDRI（高動態範圍圖像）作為背景光源：
    • 在「世界屬性（World Properties）」中，將背景設定為環境貼圖（Environment Texture），導入 HDRI。
    • 提高環境光強度，增強細節的真實感。

2. 添加景深與光暈

  1. 開啟相機的景深（Depth of Field）功能，模糊遠景，突出角色。
  2. 在「合成工作區（Compositing）」中，加入光暈效果（Glare Filter），讓場景更具吸引力。

3. 使用影片編輯器進行後期

  1. 在渲染完成後，使用 Blender 內置的「視頻序列編輯器（Video Sequencer）」進行後期調色。
  2. 增加亮度、對比度或添加特效濾鏡。

十三、解決常見問題

1. 雕刻時細節丟失

  • 啟用「動態拓撲（Dynamic Topology）」或提高網格解析度（Voxel Size 設為更小值）。
  • 檢查是否應用了細分曲面修改器。

2. UV 拉伸問題

  • 確保 UV 展開時選對了展開方式（如 Unwrap 或 Smart UV Project）。
  • 在 UV 編輯器中使用「平均拉伸（Average Island Scale）」功能，均勻縮放 UV 峰值。

3. 貼圖無法正確顯示

  • 確認貼圖節點已正確連接至材質。
  • 在「材質預覽模式」檢查是否 UV 對應正確。

十四、優化導出至遊戲引擎

 

        1. 檢查模型限制

          • 多邊形數量適合遊戲需求（例如 5k-50k 範圍）。
          • 刪除隱藏面（Invisible Faces），減少不必要的幾何。

        2. 貼圖打包

          • 使用 Blender 的「貼圖烘焙（Bake）」功能，將多層材質整合成單一貼圖。
          • 常用貼圖類型包括 Diffuse（漫射）、Normal（法線）和 Ambient Occlusion（環境光遮擋）。

        3. 選擇導出格式

          • 遊戲常用格式：FBX 或 GLTF。
          • 在導出時選擇「包括動畫（Include Animation）」和「包含貼圖（Include Textures）」選項。

BlenderCH16插件

1. 插件安裝：如何在 macOS 上啟用官方和第三方插件

步驟：

  1. 打開 Blender。
  2. 在畫面左上角的 Edit，點選 Preferences。
    • 英文界面：Edit > Preferences。
  3. 在 Preferences 視窗中，選擇上方的 Add-ons 標籤。
    • 英文界面：Preferences 窗口 > Add-ons。

插件安裝：

  1. 點選 Install 按鈕，選擇你下載的插件 .zip 檔案。

    • 英文界面：Install 按鈕。

  2. 插件安裝後會顯示在 Add-ons 列表中，勾選該插件旁邊的 Enable 方框來啟用。

    • 英文界面：勾選 Enable（啟用插件）。

  3. 點選左下角的 Save Preferences 來保存你的設置。

    • 英文界面：Save Preferences。

2. 推薦插件：如 Node Wrangler、AutoRig 等

Node Wrangler

  • 安裝與啟用：
    1. 打開 Preferences > Add-ons。
    2. 搜尋 Node Wrangler 並勾選啟用。
      • 英文界面：Preferences > Add-ons > Node Wrangler（勾選啟用）。
  • 主要功能：
    • Ctrl + Shift + 左鍵單擊：快速查看節點輸出。
    • Ctrl + T：快速添加 Mapping 和 Texture Coordinate 節點。
    • Ctrl + Shift + D：複製並自動連接節點。

AutoRig Pro

  • 安裝與啟用：
    1. 下載並安裝 AutoRig Pro 插件（通常是 .zip 檔案格式）。
    2. 在 Preferences > Add-ons 中安裝並啟用。
      • 英文界面：Preferences > Add-ons > AutoRig Pro（勾選啟用）。

其他推薦插件：

  • Hard Ops / BoxCutter：提供強大的硬邊建模工具，提升建模效率。
  • BlenderKit：提供免費和付費的 3D 模型、材質、貼圖等資源，安裝後可以直接在 Blender 裡使用。

3. 定制快捷鍵：如何根據個人習慣調整快捷鍵設置

進入快捷鍵設置：

  1. 打開 Preferences > Keymap。
    • 英文界面：Preferences > Keymap。

設定快捷鍵：

  1. 在 Keymap 視窗中，可以根據不同的操作模式選擇設置快捷鍵。

    • 例如：3D View > Object Mode。
    • 英文界面：3D View > Object Mode。

  2. 點擊命令旁邊的快捷鍵欄位，按下你想設定的新快捷鍵。

    • 例如：將 Grab/Move 的快捷鍵 G 改為 M。
    • 英文界面：Grab/Move（點擊並設置快捷鍵）。

  3. 完成後，點選 Save Preferences 保存設置。

    • 英文界面：Save Preferences。

搜索功能：

  • 在 Keymap 視窗的上方有一個 Search 框，你可以輸入關鍵字（例如：select、move），快速找到並修改相關快捷鍵。
    • 英文界面：Search 框。

重置快捷鍵：

  • 如果你想恢復預設的快捷鍵配置，點選 Restore 按鈕。
    • 英文界面：Restore 按鈕。

小結：

  • Preferences（偏好設置）是管理所有插件與快捷鍵設置的地方。
    • 在 macOS 系統中，開啟 Blender，從菜單欄選擇 Blender > Preferences 進入設置頁面。
    • 插件安裝後可以在 Add-ons 標籤下啟用，快捷鍵設置則在 Keymap 標籤下調整。

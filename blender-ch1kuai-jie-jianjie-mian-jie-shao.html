BlenderCH1安裝

1.1 認識 Blender

什麼是 Blender？
Blender 是一款功能非常強大的 3D 電腦圖形軟體，它完全免費，還是開源的，意思就是任何人都可以用它，無論是做個人作品還是商業項目都沒問題。Blender 的用途很廣，包括：

  • 3D 建模：設計角色、場景、建築模型或產品。
  • 動畫製作：做角色動畫、鏡頭運動或效果模擬。
  • 材質與渲染：創建超逼真的材質和高品質靜態圖或動畫。
  • 視覺特效（VFX）：用來做影片中的特效合成。
  • 2D 動畫與繪圖：用 Grease Pencil 工具手繪風格的動畫或圖案。

Blender 的優點有哪些？

  • 完全免費：不管是剛開始學習還是用來做專業項目，都不用花錢。
  • 支援多平台：macOS、Windows、Linux 都可以用。
  • 資源豐富：有超多教學資源、範例和免費的插件可以下載。（最後會附上我覺得好用的資源）

1.2 在 macOS 上下載和安裝 Blender

第一步：檢查你的電腦
在安裝之前，先確保你的 macOS 和硬體符合 Blender 的需求：

  • 系統需求：
    • macOS 10.13 (High Sierra) 或更新版本（建議最新版本）。
    • 如果你用的是 Apple Silicon（例如 M1、M2），選對應的版本。
  • 硬體建議：
    • 處理器：多核心 CPU。
    • 顯示卡：支援 OpenGL 3.3 或以上版本。
    • 記憶體：最少 8GB，建議 16GB 或更多。

第二步：下載 Blender

  1. 開啟 Blender 官方網站。
  2. 進入首頁後，點選「Download Blender」。系統會自動判斷你的作業系統，顯示適合 macOS 的版本。
  3. 如果你的 Mac 是 Apple Silicon，記得選擇 ARM 架構（Apple Silicon 版本） 的下載連結。
  4. 點擊下載，等待完成（檔案大小約 200MB 左右）。

第三步：安裝 Blender

    1. 打開下載的 .dmg 檔案。
    2. 把 Blender 圖示拖到「應用程式」資料夾。
    3. 如果第一次開啟 Blender 時出現安全提示（無法驗證開發者），按以下步驟解決：
      • 打開「系統偏好設定 > 安全性與隱私 > 通用」。
      • 在下方選擇「仍然開啟」。

BlenderCH14動畫技巧

1. 形狀鍵（Shape Keys）

形狀鍵通常用來控制物體或角色的變形，像是角色的表情變化或物體的變形動畫。

功能介紹：

  • 透過形狀鍵，你可以精細控制物體的頂點，做出像是表情、變形或其他細節動畫。

設置步驟：

  1. 選擇物體：首先選擇你想要設置形狀鍵的物體。
  2. 進入屬性面板：在右邊的 Properties Panel，選擇 Object Data Properties（物體數據屬性），圖示像是綠色的三角形。
  3. 新增形狀鍵：在 Shape Keys 面板裡點擊 + 來新增一個形狀鍵。
  4. 編輯形狀鍵：選擇基礎形狀（Basis），接著進行頂點編輯，創造不同的形狀變化。
  5. 插入關鍵幀：調整形狀鍵的數值（通常是0到1），然後在 Timeline 裡插入關鍵幀。

2. 約束（Constraints）

約束用來讓物體的某些屬性跟其他物體的屬性有所依賴，避免你每次都手動設置關鍵幀，可以讓動畫變得更靈活。

功能介紹：

  • 約束讓物體的動作可以跟隨其他物體的屬性，例如相機跟隨物體，或是物體跟隨其他物體的移動。

設置步驟：

  1. 選擇物體：選擇你要加約束的物體。
  2. 進入屬性面板：在右側的 Properties Panel 中，選擇 Object Constraints（物體約束），圖示像一個鏈條。
  3. 添加約束：點擊 Add Object Constraint（添加物體約束），然後選擇你需要的約束類型（例如 Follow Path 或 Track To）。
  4. 設置約束屬性：設置約束的參數，例如選擇目標物體、調整方向或偏移量等。

3. 驅動器（Drivers）

驅動器讓你可以利用一個物體的屬性來驅動另一個物體的屬性，這樣可以自動化一些動畫效果。

功能介紹：

  • 驅動器可以讓你根據一個物體的屬性變化來控制另一個物體的屬性，實現像是同步旋轉、變形等效果。

設置步驟：

  1. 右鍵屬性：在 Properties Panel 中，對你要設置驅動器的屬性（像是縮放、位置等）右鍵點擊。
  2. 添加驅動器：選擇 Add Driver（添加驅動器）。
  3. 進入圖形編輯器：打開 Graph Editor（圖形編輯器），然後在這裡編輯驅動器的來源物體和它如何影響目標物體。
  4. 設置驅動來源：在 Graph Editor 裡設置驅動的來源，像是利用物體的旋轉來控制它的縮放。

4. 非線性動畫（NLA）

非線性動畫（NLA）編輯器讓你能夠將不同的動畫片段（例如角色的行走、跳躍等）結合起來，創造更複雜的動畫。

功能介紹：

  • NLA編輯器可以讓你將多個動畫片段按時間順序或重複播放，方便管理多段動畫。

設置步驟：

  1. 進入NLA編輯器：在Blender界面的底部區域，切換到 NLA Editor（非線性動畫編輯器）。
  2. 推送動作：選擇物體，然後點擊 Push Down（推送下去），將物體的動作推送到NLA編輯器。
  3. 添加動作條目：在NLA編輯器中點擊 Add Action Strip（新增動作條目），選擇需要的動作。
  4. 管理動畫層次：你可以在NLA編輯器中調整每個動畫的開始和結束時間，控制它們的播放順序，甚至設置過渡效果。

5. 粒子系統（Particle System）

粒子系統用來創建自然現象動畫，像是煙霧、火焰、雨水等，這樣的動畫可以讓你的場景更生動。

功能介紹：

  • 粒子系統可以用來模擬煙霧、火焰、下雨等效果，這些效果通常用於自然場景。

設置步驟：

  1. 選擇物體：選擇一個物體，然後打開 Properties Panel。
  2. 進入粒子屬性：選擇 Particle Properties（粒子屬性），這個圖示像是一個小圓球。
  3. 創建粒子系統：點擊 + 創建新的粒子系統。
  4. 調整粒子屬性：設置粒子的數量、速度、生命周期等參數，然後可以在時間軸中插入關鍵幀來改變粒子的行為。

6. 相機動畫（Camera Animation）

為相機設置動畫可以讓你的動畫場景更具動感，尤其是當需要移動相機來追蹤物體或改變視角時，這是必須的。

功能介紹：

  • 相機動畫能夠幫助你設置動畫中的鏡頭運動，實現動態視角。

設置步驟：

  1. 選擇相機：選擇3D視窗中的相機物體。
  2. 進入相機屬性：在 Properties Panel 中，選擇 Camera（相機）面板，這個圖示像一個小相機。
  3. 設置位置與旋轉：調整相機的位置和旋轉，並在 Timeline 上插入關鍵幀。
  4. 使用Track To約束：你可以為相機設置 Track To（追蹤目標）約束，讓相機始終對準指定的物體，這樣相機就能跟隨物體運動。

7. 音效同步（Synchronizing Sound with Animation）

音效與動畫的同步對於提升動畫的真實感和效果非常重要，Blender也有提供相關功能來處理這一部分。

功能介紹：

  • 你可以將音效加入動畫中，並且和動畫的事件（像是跳躍、碰撞等）進行同步。

設置步驟：

  1. 導入音效：進入 Video Sequence Editor（視頻序列編輯器），點擊 Add > Sound（音效），然後選擇你的音效檔案。
  2. 調整音效時間：在 Timeline 上調整音效的播放時間，讓它跟動畫的事件對齊。
  3. 檢查同步：播放動畫並確認音效與動畫同步。

Blender 初次打開的初始設定（Splash Screen）

每次打開 Blender，都會看到一個「歡迎畫面」（Splash Screen）。它有幾個主要功能，你可以用來快速設置工作環境：

1. 選擇專案模板

  • General（一般用途）：最常用的模板，適合建模、動畫和其他基礎操作。
  • 2D Animation（2D 動畫）：專門為 2D 動畫設計，使用 Grease Pencil 工具。
  • Sculpting（雕刻）：適合進行角色雕刻或高細節模型處理。
  • VFX（視覺特效）：用來做特效合成或追蹤。
  • Video Editing（影片剪輯）：提供簡單的影片剪輯工具。

你可以根據需求選擇適合的模板，選完後就進入對應的工作環境。

2. 快速開啟最近的檔案

畫面下方會列出你最近打開的檔案。如果之前有做專案，直接點就能快速回到上次的進度。

3. 初次設定選項

右上角的「New File > Preferences」可以調整一些基本設定，建議新手特別注意以下幾點：

  1. 語言設定

    • 預設是英文介面，習慣中文的話，可以到「Edit > Preferences > Interface > Language」選擇中文。（但是是簡體的樣子⋯⋯，建議用英文舊好，未來跟別人工作會比較好溝通！）

  2. 快捷鍵方案

    • 預設是 Blender 的快捷鍵，但如果你來自其他軟體（例如 Maya），可以切換到對應的方案。

  3. 顯示單位

    • 建議設置成「公制」（Metric），方便計算長度、尺寸。

  4. 觸控板支援

    • 如果用 MacBook 的觸控板，可以啟用「Emulate 3 Button Mouse」，讓操作更直覺。

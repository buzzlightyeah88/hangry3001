BlenderCH19粒子、流體、布料

1. 粒子系統基礎

1.1 開啟粒子系統

  1. 選擇物體：

    • 在 3D 視窗中選擇一個物體，這個物體會作為粒子的發射源。
    • 例如，可以選擇一個立方體或球體。

  2. 進入粒子系統屬性：

    • 在右側面板中，找到一個名為粒子屬性（Particles Properties）的圖標，這個圖標看起來像是一個小圓圈和尾巴的形狀，位於一系列屬性選項的中間部分。

  3. 新增粒子系統：

    • 點選**+ New** 按鈕，會創建一個新的粒子系統，並且此時會出現粒子屬性面板。

1.2 設定粒子系統

  1. 選擇粒子類型：

    • 在Type欄位中，選擇粒子系統的類型：
      • Emitter（發射器）：粒子會被發射並且向外散開，通常用於煙霧、火焰等效果。
      • Hair（毛發）：粒子會固定在物體上，通常用來模擬頭髮或草等。

  2. Number（粒子數量）：

    • 在Number欄位中，設置要生成的粒子數量。你可以手動調整此數值，通常煙霧或火焰需要更多的粒子。

  3. Frame Start / End（發射起始與結束幀）：

    • 設定粒子生成的起始與結束時間。這樣可以控制粒子生成的時間範圍。

  4. Lifetime（粒子生命週期）：

    • 設定每個粒子的生存時間，通常用幀數來表示。例如，如果你希望粒子在30幀後消失，設置為30。

  5. Emit From（發射位置）：

    • 設定粒子生成的區域。選擇Faces（面）會使粒子從物體的表面發射，選擇Volume（體積）則會從物體的內部發射粒子，選擇Vertices（頂點）會從物體的頂點發射粒子。

  6. Initial Velocity（初始速度）：

    • 在Velocity選項下，你可以調整粒子的速度：
      • Normal：設定粒子沿著物體表面的法線方向的速度。
      • Random：設定隨機的速度，使粒子在不同的方向上發射。

  7. Render（渲染顯示）：

    • 在粒子屬性面板的Render區域，你可以設定粒子的顯示樣式：
      • Halo（光暈）：使粒子顯示為小圓點，常用於煙霧等效果。
      • Object（物體）：讓每個粒子渲染成指定的物體（例如小球、立方體等）。
      • Group（群組）：使用一組物體作為粒子渲染。

1.3 進階效果：煙霧與火焰

  1. 煙霧：

    • 點選Render欄位，選擇Halo來讓粒子顯示為光暈。
    • 在Physics（物理）面板中，選擇Fluid（流體）選項來設定煙霧源。
    • 在Smoke（煙霧）模組中，選擇Flow Type為Smoke，並設置Flow Behavior（流動行為）為Inflow，這樣可以讓粒子形成煙霧效果。

  2. 火焰：

    • 在Render欄位中，選擇Object，並將火焰對應的物體設置為粒子物體。
    • 在Physics面板中，選擇Fluid，並將Flow Type設置為Fire，這樣粒子就會模擬出火焰。

2. 流體模擬

2.1 開啟流體模擬

  1. 選擇物體：

    • 在 3D 視窗中選擇一個物體，這個物體將作為流體模擬的容器或流體源。

  2. 開啟流體屬性：

    • 在右側面板中選擇物理屬性（Physics Properties），並選擇Fluid（流體）選項。

2.2 設定流體模擬

  1. Domain（流體範圍）：

    • 設定一個物體作為Domain（流體範圍），這個物體會限制流體的模擬範圍。常見的選擇是使用立方體作為容器。

  2. Fluid Type（流體類型）：

    • 設定流體的類型，可以選擇：
      • Liquid（液體）：用來模擬水流等液態物質。
      • Gas（氣體）：用來模擬煙霧或火焰。

  3. Flow（流動源）：

    • 在Flow Type欄位中，設定流體源：
      • Liquid（液體）：模擬水流。
      • Gas（氣體）：模擬煙霧。
    • 設定流體源的形狀，例如使用物體的表面（Mesh）或體積（Volume）來發射流體。

  4. Resolution Divisions（解析度分割）：

    • 在Domain設置中，調整Resolution Divisions來控制流體模擬的細節。數值越大，模擬效果越細緻，但會需要更多計算資源。

  5. Viscosity（粘度）：

    • 在Flow設定中，調整流體的粘度。高粘度會使流體像蜂蜜或油漬一樣流動緩慢，低粘度則像水一樣流動快速。

  6. Surface Tension（表面張力）：

    • 設定流體的表面張力，會影響水滴的形狀和流動行為。

3. 布料與剛體模擬

3.1 布料模擬

  1. 選擇布料物體：

    • 在 3D 視窗中選擇一個物體，通常是用來模擬布料的物體。最常見的是選擇一個平面物體（如方形或圓形）。

  2. 開啟布料物理：

    • 在右側面板中選擇物理屬性（Physics Properties），並選擇Cloth（布料）選項。

3.2 設定布料模擬

  1. 質量：

    • 在Cloth屬性面板中，設定Mass（質量）。較小的質量會讓布料更輕，較大的質量則會讓布料下垂得更快。

  2. 張力與剛性：

    • Stiffness（剛性）：控制布料的彈性，數值越大，布料越硬。
    • Bending（彎曲）：控制布料彎曲的柔軟度，數值越小，布料越柔軟。

  3. 風與重力：

    • 在Field Weights部分，調整Wind（風力）與Gravity（重力）的影響，來模擬風對布料的影響。

  4. 碰撞：

    • 在Collision區域中，開啟布料與其他物體的碰撞檢測。設定Thickness（厚度）來避免物體穿透布料。

3.3 剛體模擬

  1. 選擇剛體物體：

    • 在 3D 視窗中選擇一個物體，這個物體會進行剛體模擬。

  2. 開啟剛體物理：

    • 在右側面板中選擇物理屬性（Physics Properties），並選擇Rigid Body（剛體）選項。

3.4 設定剛體模擬

  1. Type（類型）：

    • 設定為Active（主動）或Passive（靜態）。
      • Active：物體會根據物理模擬運動，並與其他物體發生碰撞。
      • Passive：物體保持靜止，但仍與主動物體發生碰撞。

  2. Mass（質量）：

    • 設定物體的質量，影響物體的運動。

  3. Friction（摩擦力）：

    • 設定物體與其他物體之間的摩擦力。

  4. Restitution（彈性）：

    • 設定物體的反彈程度，數值越高，物體反彈越強。
